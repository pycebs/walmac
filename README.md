# walmac
macOS color scheme creator for ITerm and Terminal

## Requirements
1. Python 3
2. ImageMagick - `brew install imagemagick`
3. walmac

## Usage
```
$ chmod +x walmac.py
$ ./walmac -i /path/to/wallpaper.jpg
```
